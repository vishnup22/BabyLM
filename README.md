BabyLM 2026 Baselines
=====================

This repository contains the code used to train the baselines for the 2026 iteration of the BabyLM Challenge. Refer to `strict-gpt2` for code for the naive GPT-2 baseline for Strict/Strict-Small/Multilingual tracks and `strict-interaction` for the past year's Interaction track baseline adapted to the Strict/Strict-Small tracks.
